What is a div element?

The div element is like a box that groups parts of a webpage together.

What is one way you can identify or "target" a div element?

To style a specific div, you use its class name. When you use a class in CSS, it changes every div that has that same class which lets you use the same style on different parts of a website.

Why would you use a div element?

You would use a div element to organize and structure your webpage.

What is a border property?

The border property allows you to style the border around an element.

What are some of the different border properties?

Borders, Border width, Border color, Border sides, & Rounded borders

What is the difference between margins and padding?

Margins and padding both create space but in different places. Padding is the space inside an element while Margins is the space outside an element.
